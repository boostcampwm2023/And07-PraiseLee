## ✨ 프로젝트 소개

- 마인드맵을 사용하여 아이디어를 시각적으로 표현해보세요
- 협업을 원하는 친구나 동료를 초대하는 것도 가능합니다
- 간편하게 마인드맵을 생성하고, 다른 사용자들과 실시간으로 공동 편집을 진행할 수 있어요
